Info®360の紹介

WEBで建物情報を一元管理

  • 点群表示

    ✓ 軽快な操作感を実現
    ✓ 増築・改修工事の事前検討に活用

  • 点群表示

    ✓ 都市などの広範囲な点群データも表示可能
    ✓ 都市計画や防災計画のシミュレーションに活用

  • アノテーション

    ✓ 画面内に注釈を自由に登録
    ✓ 写真や指示書等のデータを添付

  • 図面表示

    ✓ PDF図面のインポートが可能
    ✓ 建物モデルと重ね合わせて検討

  • 計測機能

    ✓ 長さ・高さ・角度等のあらゆる必要寸法を計測
    ✓ 出隅・入隅を正確にスナップ

  • 視点切替

    ✓ パース投影·平行投影の切替が可能
    ✓ 真上・真横の様々なプリセット視点を用意

  • モデル表示

    ✓ IFCデータをインポート
    ✓ BIMで作成した鉄筋や躯体などを表示可能

Info®360の機能

他の追随を許さない豊富な機能

  • 点群表示 建築物や工作物の点群データを表示することが可能
  • 3Dモデル表示 IFCデータ変換によりBIMモデルの表示が可能
  • アノテーション 画面内に注釈を登録し担当者や資料の添付が可能
  • レイヤー表示 要素ごと(建物·図面·アノテーション等)の表示が可能
  • PDF図面表示 PDF図面の表示や建物との重ね合わせが可能
  • 表示容量 1プロジェクトで約1TBの表示が可能
  • クリップ範囲表示 必要な範囲のみを抽出し表示することが可能
  • ポジション モデルの水平垂直移動·回転角度等の位置調整が可能
  • 視点切替 バードビューや平面立ピューなど視点切替機能を搭載
  • 投影法切替 パース投影·平行投影の切替えが可能
  • 点群表示精度切替 Standard~High-Qualityの表示切替え
  • URLリンク共有 関係者への共有がURL送付にて簡単に行える
  • 計測 距離·角度·座標等の計測機能を搭載
  • ダブルクリック移動 目的の箇所をダブルクリックしスムーズな移動を実現
  • 写真登録 アノテーション内に撮影した写真の登録が可能
  • S造·鉄骨 BIMで作成した鉄骨躯体モデルをの表示が可能
  • RC造·鉄筋 BIMで作成した鉄筋モデルの表示が可能
  • 木造·構造金物 BIMて作成した木造モデルの表示が可能
  • BCF チャットを中心としたコミュニケーション機能を搭載
  • 担当者登録 アノテーション内に担当者や作業種別等を記載出来る

充実のサポートもお試し下さい

  • 測量業務

    当社スタッフが現場まで出張し測量作業を行います。 計測規模は住宅・高層ビル・大型複合施設·競技場(ドーム)・アリーナ・街並み・都市と、あら中る規模に対応します。


  • 点群合成処理

    測量した点群デ-タの合成処理作業も当社スタッフが一気通貫で賄います。
    計測距離10mにおいて±4mmの誤差(※1)に納まる高精度の機器を用いるので、現場との誤差が限りなく少ない成果品を納品致します。

  • BIMモデル作成

    計測した点群データからBIMモデルを作成し納品することも可能です。BIMモデルがあれば改修や增策工事計画における事前検討の効率が格段に高くなり、工期短縮やミスの防止なと、無駄なコストを抑えることに寄与します。
